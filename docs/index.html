MultiRoblox

Run two Roblox clients at the same time on one PC.

⇩ Download for Windows Installer (auto-update)

Portable .exe — just run it. Windows 10/11. See the latest release.

One click

Open the app and press “Launch 2 Clients”. Two Roblox windows open, ready for two accounts.

Live status

See exactly which clients are running and stop them all with a single button.

Safe & simple

No files patched, no admin rights. It just holds Roblox’s single-instance lock while open.

How to use

  1. Download and run MultiRobloxController.exe.
  2. Click Launch 2 Clients. The classic player and the Microsoft Store client both open.
  3. Sign into a different Roblox account in each window and join a game.
  4. Keep the controller window open while you play. Use Stop All Clients when finished.

Install from PowerShell

Prefer a command line? This downloads the latest release and adds shortcuts:

powershell -NoProfile -ExecutionPolicy Bypass -Command "iwr -useb https://raw.githubusercontent.com/anriltine/MultiRoblox/main/installer/Install-MultiRoblox.ps1 | iex"

Requirements

  • Windows 10 or 11.
  • The classic Roblox player (from roblox.com) and the Microsoft Store Roblox app both installed — this pairing is what allows two clients to stay open.
