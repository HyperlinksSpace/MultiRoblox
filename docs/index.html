MultiRoblox

Run as many Roblox instances as you want on one PC.

⇩ Download MultiRoblox.exe

A single .exe — no install needed. Windows 10/11. See the latest release.

One button

Press “Launch Instance” once per Roblox window you want. Every press opens another client.

Live status

See every running instance with its uptime and stop them all with a single button.

Safe & simple

No files patched, no admin rights. It just holds Roblox’s single-instance lock while open.

How to use

  1. Download MultiRoblox.exe and run it.
  2. Press Launch Instance — once for every Roblox window you want open.
  3. Sign into a different Roblox account in each window and join a game.
  4. Keep MultiRoblox open while you play. Use Stop All when finished.

Optional: install with auto-update

Want Start Menu shortcuts and automatic updates? Run this in PowerShell:

powershell -NoProfile -ExecutionPolicy Bypass -Command "iwr -useb https://raw.githubusercontent.com/HyperlinksSpace/MultiRoblox/main/installer/Install-MultiRoblox.ps1 | iex"

Requirements

  • Windows 10 or 11.
  • The Roblox player installed (from roblox.com). Keep MultiRoblox open while playing — it holds the lock that lets extra instances stay alive.
